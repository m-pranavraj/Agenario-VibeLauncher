Final Agenario Rule

A finding is not allowed into a report unless it includes at least one of:

✅ File + Line

✅ Code Snippet

✅ Runtime Screenshot

✅ Network Evidence

✅ Reproduction Steps

✅ Stack Trace

✅ Dependency Evidence

Otherwise mark it:

AI Observation
Confidence: 50%

and separate it from verified findings.

This is probably the single most important upgrade you can give your AI agent:

"Every finding must be evidence-backed. No anonymous AI claims allowed."
